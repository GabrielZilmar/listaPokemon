# Lista Pokemon
Antigo projeto feito em Jan 2020 para aprender sobre React.JS na rota de FrontEnd do camp da Ioasys.

### Instalação
 - Clone o repositório com:
    ```git clone https://github.com/GabrielZilmar/listaPokemon.git```
 - Na pasta do repositório baixe as dependências utilizando seu gerenciador de pacotes preferido. Exemplo utilizando npm:
   ```npm install```
 - Após a instalação das dependências rode a aplicação com o script start. Exemplo utilizando npm:
   ```npm start```
 - Pronto! Agora só ir no seu navegador preferido e abrir a aplicação na porta 3000 da sua localhost: http://localhost:3000.
